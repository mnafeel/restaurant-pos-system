🛡️ DATA RECOVERY (If Website Crashes)
═══════════════════════════════════════════════════════════

GOOD NEWS: Your data is SAFE!

Why:
  • Data stored in FILE on your computer
  • NOT stored in memory or cloud
  • Survives crashes, restarts, power off
  • Backed up automatically

To Recover:
  1. Just restart the server:
     cd /Users/admin/restaurant-billing-system
     node server.js
  
  2. ✅ All data loads automatically!

If Database Corrupted:
  1. Restore from backup:
     cp backups/restaurant-db-20251016.db restaurant.db
  
  2. Restart server
  
  3. ✅ Data restored!
