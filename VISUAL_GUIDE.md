## 🎯 Quick Actions

### Upload Avatar
```
1. Click name in sidebar
2. Click 📷
3. Select image
4. Done! ✅
```

### Change Avatar
```
1. Click name
2. Click 📷 (replaces old)
3. Select new image
4. Old deleted automatically ✅
```

### Remove Avatar
```
1. Click name
2. Click 🗑️
3. Confirm
4. Back to initials ✅
```

### Upload Shop Logo
```
1. Shops page
2. Click 📷 on shop
3. Select logo
4. Done! ✅
```
