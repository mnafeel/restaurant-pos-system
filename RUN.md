# 🚀 How to Run Your Restaurant POS

## ⚡ FASTEST WAY (2 Commands):

### Terminal 1 - Backend:

```bash
cd /Users/admin/restaurant-billing-system
node server.js
```

### Terminal 2 - Frontend:

```bash
cd /Users/admin/restaurant-billing-system/client
npm start
```

**That's it!** Wait 30 seconds, then open: **http://localhost:3000**
